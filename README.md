# Многопоточный проект с использованием циклической блокировки (Практическая работа 6)

## Описание

Этот проект реализует многопоточные программы для поиска элемента в массиве с использованием различных примитивов синхронизации. В проекте представлены два задания:

- **Задание 1** (Легкий уровень): Поиск первого вхождения заданного значения в массиве с использованием **циклической блокировки (spinlock)**.
- **Задание 2** (Средний уровень): Поиск всех вхождений заданного значения в массиве с использованием **циклической блокировки (spinlock)**.

Программа использует библиотеку **pthread** для многозадачности и синхронизации потоков.

## Структура проекта

- **task1.c**: Код для выполнения задания 1 (поиск первого вхождения с использованием **spinlock**).
- **task2.c**: Код для выполнения задания 2 (поиск всех вхождений с использованием **spinlock**).
- **CMakeLists.txt**: Файл конфигурации для сборки проекта с использованием CMake.
- **README.md**: Этот файл с описанием проекта.
- **.gitignore**: Файл для игнорирования временных и скомпилированных файлов.

## Требования

Для сборки и запуска проекта вам потребовались:

- **MSYS2***.
- **Компилятор GCC**.
- **CMake** для сборки проекта.
## Запуск
- ./task1
- ./task2
